  ## Description

  Generate a README.md file based on a couple of questions prompted though the CLI.

  ## Installation

  To install all necessary files, use the following command on the CLI;
  
  ```
  npm i
  ```

  ## Tests

  To run tests, use the following command on the CLI;
  
  ```
  npm test
  ```
